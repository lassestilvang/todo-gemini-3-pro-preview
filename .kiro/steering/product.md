# Product Overview

Todo Gemini is an AI-powered daily task planner designed for speed, simplicity, and great UX.

## Core Features

- Task management with lists, labels, priorities, due dates, and deadlines
- Smart views: Inbox, Today, Next 7 Days, Upcoming, Calendar
- Subtasks and task dependencies (blocking relationships)
- Recurring tasks with RRule support
- Task templates with variable substitution
- Smart tagging: AI-assisted list/label suggestions based on task title
- Focus mode for individual task concentration
- Search across all tasks

## Gamification

- XP system with levels (completing tasks awards XP)
- Achievements based on task completion counts and streaks
- Daily streaks tracking
- Confetti animations on task completion

## Additional Features

- Activity log tracking all task changes
- Habits tracking with completion history
- Energy level and context tagging (computer, phone, errands, etc.)
- Time estimates for tasks
- PWA support for mobile installation
- Multiple theme support (light, dark, glassmorphism, neubrutalism, minimalist)
